Simply Keep track of your site's visitor. It’s fast and light.

== Description ==

Keep track of your site visitor's details like Country, IP, Referrer, User Agent, visit time. Its very light plugin that doesn't effect your page loading speed.Its simple and easy.


= Key Feature =

* Fast and light
* Very simple to use
* Not effect page loading speed
* Tracks visitor country, ip, referrer, user agent, visiting time


== Installation ==

Using the WordPress dashboard

1. Login to admin panel
2. Go to Plugins
3. Select Add New
4. Search Simple visitor stat
5. Select Install Now
6. Select Activate Plugin

Manual

1. Download the plugin and extract the files
2. Upload the directory "simple-visitor-stat" to your wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Check the "Visitor stat" Tab created by this plugin for view visitor stat

That's all. You dont need to do anything else. Enjoy!
